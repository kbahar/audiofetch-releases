# AudioFetch

A standalone desktop application for downloading audio files from Soundgasm with proper filenames.

## Download

**[Download Latest Version](https://github.com/kbahar/audiofetch-releases/releases/latest)**

| File | Description |
|------|-------------|
| `AudioFetch Setup X.X.X.exe` | Windows installer - installs to your system |
| `AudioFetch X.X.X.exe` | Portable version - no installation needed, just run |

## Features

- **Manual Download** - Paste Soundgasm URLs to download audio files with proper names
- **User Profile Scan** - Search Reddit and other sources for a user's audio files
- **Batch Download** - Download multiple files as a ZIP archive
- **No Rate Limiting** - Runs locally on your machine, no server restrictions

## How to Use

1. Download and run AudioFetch
2. **Manual Mode**: Paste Soundgasm URLs (one per line) and click "Parse URLs"
3. **Scan Mode**: Enter a Soundgasm user profile URL to search for their content
4. Download individual files or all at once as a ZIP
